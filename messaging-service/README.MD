# Serviço de Mensageria

Este serviço ficará responsável por centralizar a comunicação assíncrona entre os sistemas da aplicação.

Sua função será receber, organizar e distribuir mensagens entre os serviços, permitindo o envio de eventos, processamento em fila e desacoplamento entre as partes do sistema.

Na prática, ele servirá como a camada de mensageria do projeto, apoiando integrações internas e fluxos que não precisam acontecer de forma imediata.
